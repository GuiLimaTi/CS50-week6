## Projects

### readability.py
Implements a readability analyzer using the Coleman-Liau index.

- Counts letters, words, and sentences from user input
- Calculates reading grade level
- Demonstrates string processing and mathematical computation

---

### cash.py
Solves the minimum coin problem using a greedy algorithm.

- Converts dollars to cents to avoid floating-point errors
- Calculates minimum number of coins (quarters, dimes, nickels, pennies)
- Demonstrates optimization using greedy strategies

---

### dna.py
Identifies individuals based on DNA STR sequence matching.

- Reads structured data from CSV files
- Processes DNA sequences from text files
- Computes longest consecutive STR matches
- Compares results against a database to find a match

---

### mario.py
Generates a pyramid structure based on user input.

- Uses loops and conditionals
- Demonstrates control flow and pattern building
